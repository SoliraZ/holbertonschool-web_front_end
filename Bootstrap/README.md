# Bootstrap Exercises

This directory contains example HTML files demonstrating how to use [Bootstrap 4](https://getbootstrap.com/) for responsive web layouts and styling.

## Features

- Responsive grid layouts using Bootstrap 4
- Examples of columns, offsets, and display order
- Embedded custom styles for demonstration
- Out-of-the-box, mobile-first responsiveness
